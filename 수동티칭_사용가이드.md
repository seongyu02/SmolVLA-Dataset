# 🎓 수동 티칭 로깅 프로그램

로봇을 **수동으로 움직이면서** 실시간으로 관절값과 좌표값을 기록하는 프로그램입니다.

---

## 🎯 사용 시나리오

이 프로그램은 다음과 같은 경우에 유용합니다:

1. **티칭 작업**: 로봇을 원하는 경로로 직접 끌고 가면서 경로 기록
2. **경로 학습**: 수동으로 최적 경로를 찾고 데이터로 저장
3. **데모 수집**: 로봇 동작 시연을 데이터로 기록
4. **궤적 학습**: 기계 학습을 위한 시연 데이터 수집
5. **품질 검사**: 수동 동작 시 로봇의 정밀도 확인

---

### 1단계: 로봇 준비

로봇을 **수동으로 움직일 수 있는 상태**로 만드세요:

#### 방법 1: DobotStudio Pro에서 Disable
```
DobotStudio Pro 실행
→ 로봇 연결
→ Enable 버튼 클릭하여 Disable 상태로 전환
```

```
→ 로봇을 자유롭게 움직일 수 있음
```


## ⌨️ 키보드 명령

### `r` - 로깅 시작 (Record)

로봇의 위치를 실시간으로 기록하기 시작합니다.

**실행 화면:**
```
====================================================================
📝 로깅 시작: Session_1
====================================================================
💡 로봇을 원하는 대로 움직이세요.
   's' 키를 누르면 로깅이 중지되고 저장됩니다.
====================================================================

📝 로깅 중... (240개 데이터 수집)
```

**이제:**
- 로봇을 원하는 경로로 천천히 움직이세요
- 모든 동작이 실시간으로 기록됩니다 (20Hz)
- 초당 20개의 데이터 포인트 수집

---

### `s` - 로깅 중지 및 저장 (Stop)

로깅을 중지하고 CSV 파일로 저장합니다.

**실행 화면:**
```
====================================================================
📝 로깅 종료 (총 480개 데이터 수집)
====================================================================

💾 로그 저장 완료: logs\manual_teaching_20260122_153045.csv
   총 480개 데이터 포인트
```

**저장 위치:**
- `logs/manual_teaching_YYYYMMDD_HHMMSS.csv`

---

### `p` - 현재 위치 확인 (Position)

로봇의 현재 위치를 실시간으로 확인합니다.

**실행 화면:**
```
----------------------------------------------------------------------
📍 현재 로봇 위치:
   관절각: J1=-70.197°, J2=-27.691°, J3=-4.123°
           J4=-17.403°, J5=93.926°, J6=29.172°
   좌표계: X=27.57mm, Y=-302.40mm, Z=481.98mm
           Rx=-144.11°, Ry=21.33°, Rz=179.05°
   로봇 모드: 4
----------------------------------------------------------------------
```

---

### `h` - 도움말 (Help)

사용 가능한 명령어 목록을 다시 표시합니다.

---

### `q` - 프로그램 종료 (Quit)

프로그램을 종료합니다.

**주의:**
- 로깅 중일 때는 먼저 `s`를 눌러 저장해야 합니다.

## 📊 사용 예시

### 시나리오 1: 단순 경로 기록

```
1. python manual_teaching_logger.py 실행
2. 'r' 키 입력 → 로깅 시작
3. 로봇을 A 위치 → B 위치로 천천히 이동
4. 's' 키 입력 → 로깅 중지 및 저장
5. 'q' 키 입력 → 종료
```

**결과:**
- `logs/manual_teaching_YYYYMMDD_HHMMSS.csv` 생성
- A→B 경로의 모든 중간 위치 기록됨

---

### 시나리오 2: 복잡한 작업 경로

```
1. python manual_teaching_logger.py 실행
2. 'r' 키 입력 → Session_1 로깅 시작
3. 로봇으로 복잡한 작업 동작 시연
4. 's' 키 입력 → 저장
5. 'r' 키 입력 → Session_2 로깅 시작
6. 다른 작업 동작 시연
7. 's' 키 입력 → 저장
8. 'q' 키 입력 → 종료
```

**결과:**
- 2개의 세션이 1개 CSV 파일에 저장됨
- 각 세션은 `session_name` 컬럼으로 구분

---

### 시나리오 3: 위치 확인하며 기록

```
1. python manual_teaching_logger.py 실행
2. 로봇을 원하는 위치로 이동
3. 'p' 키 입력 → 현재 위치 확인
4. 위치가 맞으면 'r' 키 입력 → 로깅 시작
5. 다음 위치로 이동
6. 's' 키 입력 → 저장
7. 'p' 키 입력 → 최종 위치 확인
8. 'q' 키 입력 → 종료
```

## 💡 활용 방법

### 1. 경로 재현

수동으로 움직인 경로를 자동으로 재현:

```python
import csv

with open('logs/manual_teaching_20260122_153045.csv', 'r') as f:
    reader = csv.DictReader(f)
    for row in reader:
        # 기록된 관절각으로 이동
        robot.MovJ(
            float(row['j1']), float(row['j2']), float(row['j3']),
            float(row['j4']), float(row['j5']), float(row['j6']),
            1, v=30, a=30
        )
        sleep(0.05)  # 기록 속도와 동일
```

### 2. 경로 최적화

수동 경로를 분석하여 중요한 포인트만 추출:

```python
import csv

# 10번째 데이터마다 추출 (다운샘플링)
with open('logs/manual_teaching.csv', 'r') as f:
    reader = csv.DictReader(f)
    key_points = [row for i, row in enumerate(reader) if i % 10 == 0]

# 주요 포인트만 저장
print(f"원본: {len(all_data)}개 → 최적화: {len(key_points)}개")
```

### 3. 경로 비교

여러 번의 시연을 비교하여 일관성 확인:

```python
# Session_1과 Session_2의 경로 비교
session1 = [row for row in data if row['session_name'] == 'Session_1']
session2 = [row for row in data if row['session_name'] == 'Session_2']

# 차이 계산
differences = compare_trajectories(session1, session2)
```

### 4. 기계 학습 데이터

수동 시연을 학습 데이터로 사용:

```python
# 시연 데이터를 학습용으로 변환
X = []  # 입력: 현재 위치
y = []  # 출력: 다음 위치

for i in range(len(data) - 1):
    current = data[i]
    next_pos = data[i + 1]
    
    X.append([current['j1'], current['j2'], ...])
    y.append([next_pos['j1'], next_pos['j2'], ...])
```

## ⚠️ 주의사항

### 1. 로봇 상태
- 로봇이 **수동으로 움직일 수 있는 상태**여야 합니다
- Disable 또는 드래그 티칭 모드 필수
- Enable 상태에서는 로봇이 움직이지 않습니다

### 2. 안전
- 로봇을 천천히 움직이세요
- 주변에 장애물이 없는지 확인
- 급격한 동작 금지

### 3. 데이터 품질
- 너무 빨리 움직이면 부정확한 경로 기록
- 천천히 일정한 속도로 움직이는 것이 좋음
- 떨림이 없도록 안정적으로 움직임

### 4. 파일 크기
- 장시간 로깅 시 파일 크기 증가
- 10분 로깅 시 약 2~3MB 예상

## 🎯 팁 & 트릭

### 팁 1: 부드러운 경로
로봇을 **천천히, 부드럽게** 움직이면 더 좋은 데이터를 얻을 수 있습니다.

### 팁 2: 여러 번 시연
같은 작업을 여러 번 시연하여 평균 경로를 구하면 더 정확합니다.

### 팁 3: 주요 포인트 확인
경로 중간에 `s`를 눌러 저장하고, `p`로 위치를 확인한 후 다시 `r`로 계속할 수 있습니다.

### 팁 4: 세션 분리
복잡한 작업은 여러 세션으로 나누어 기록하면 관리가 쉽습니다.

## 📞 문제 해결

### Q: 로봇이 움직이지 않아요
**A:** 로봇이 Disable 상태이거나 드래그 티칭 모드가 활성화되어 있는지 확인하세요.

### Q: 로깅이 시작되지 않아요
**A:** 로봇 연결 상태를 확인하세요. 피드백 포트(30004)가 정상적으로 연결되어야 합니다.

### Q: 파일이 너무 커요
**A:** 샘플링 주기를 늘리거나(예: 0.1초), 짧은 경로만 기록하세요.

### Q: 키 입력이 인식되지 않아요
**A:** 이 프로그램은 Windows 전용입니다. Linux/Mac은 지원하지 않습니다.
